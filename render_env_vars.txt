RENDER ENVIRONMENT VARIABLES
============================

Copy and paste these into your Render web service environment variables:

SECRET_KEY = django-insecure-gst-compliance-system-2025-render-deployment-secret-key-change-this-in-production

DEBUG = False

ALLOWED_HOSTS = *.onrender.com

# Optional: For PostgreSQL database (leave empty to use SQLite)
DATABASE_URL = 

# Optional: For Redis (leave empty if not using Celery)
REDIS_URL = 

============================
INSTRUCTIONS:
1. Go to your Render web service
2. Click "Environment" tab
3. Add each variable above
4. Click "Save Changes"
5. Your app will redeploy automatically
